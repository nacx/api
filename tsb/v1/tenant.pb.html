Tenant is a root of the Service Bridge object hierarchy. Each tenant is completely independent of the other with its own set of users, teams, clusters and workspaces.

The following example creates a tenant named mycompany.

apiVersion: api.tsb.tetrate.io/v1
kind: Tenant
metadata:
  name: mycompany
